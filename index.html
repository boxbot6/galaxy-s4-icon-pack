Install the icon pack (.apk) on your device and use Nova Launcher or another third-party launcher to install the whole icon pack together at one time or you can edit the individual icons by selecting this icon pack as a source within the launchers when editing individual icons. A longer explanation for how to use the icon pack:

An icon pack is a collection of icons that are designed to be used together as a set - they are stored inside an .apk wrapper (basic app) along with their resource files, reference lists and other files that define their associated app usage etc. These icon packs can be used by other apps such as launchers or used on their own.

Launchers can be used to change all sorts of things about the phones appearance from changing the icons to tweaking every aspect of your phone's interface, layout and animations. They can replace the default home screen and app drawer of a mobile device and allow users to endlessly customize the look and feel of their device's interface.

Some general notes about this icon pack:

  • This icon pack was made using the template 'icon-pack-template-basic' which you can find here together with an explanation of how to use it (you can use the template to create your own icon pack using any icons you want, either from existing icons or you can create your own icons or images as you wish).
  • The app has a min SDK/API level of 16 (Jelly Bean).
  • The icons contained in this pack are basically just .png images with transparent backgrounds.
  • The icons included in this pack are of various sizes - nowadays 512 x 512 pixel icons in .png format appears to be a common standard if you wanted to rescale them all to match each other but please note they may not then scale down properly again to work on older phones.
  • This pack has been updated to contain a lot of added alternative icons which use a suffix like (*_alt_01) - because they are all listed in named order you may have to scroll past all those to find the original icons that you want.
  • This icon pack .apk uses quite a basic code and has not been signed so you will need to allow apps from unknown sources and check the version of Android on your device (the min SDK/API level is 16 (Android 4.1 Jelly Bean)).
